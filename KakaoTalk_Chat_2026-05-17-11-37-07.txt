[To Do Dist 개선 정리]

1. UI / 사용성 문제

- 투명도 부족 → 화면을 많이 가림
- 앱 실행 시 공간 차지 큼
- 다른 작업과 병행 어려움
- 작은 사이즈 / 크기 조절 필요

[개선 방향]
- 위젯형 미니 모드
- 투명도 조절 기능
- 플로팅 UI
- 드래그 크기 조절
- 집중모드
  : 현재 할 일 1개
  : 진행률
  : 남은 시간 표시


2. 동기부여 문제

[문제]
- 오늘 도파민 부족해서 하기 싫음
- 체크아웃 후 방치
- 동기 유지 어려움

[원인]
- 단순 관리형 Todo
- 감정 흐름 고려 X

[개선]
- 감정 기반 시스템

예시 감정:
- 지침
- 무기력
- 집중됨
- 불안함

상태별 메시지:
- “오늘은 5분만 해도 충분해.”
- “완벽보다 시작이 중요해.”
- “조금이라도 하는 사람이 결국 이긴다.”


3. 동기 유지 시스템

[문제]
- 체크 후 성취감 지속 X
- 장기 목표와 현재 행동 연결 약함

[아이디어]
- 격언 / 동기 메시지 제공

예시:
- “미래는 오늘 반복한 행동으로 결정된다.”
- “동기보다 중요한 건 시스템이다.”
- “지금의 10분이 미래를 만든다.”


4. 사용자 성향

[J 유형 사용자]
- 계획형
- 생산성 추구
- 각잡고 사는 삶 선호
- 목표 달성 욕구 강함

[필요 기능]
- 체계적 관리
- 시간 최적화
- 장기 목표 추적
- 자기 통제감


5. 사용자 시나리오

[시나리오 A - 시험기간 학생]

상황:
- 여러 과목 공부 필요

문제:
- 시간 배분 어려움
- 우선순위 혼란

필요 기능:
- 시간 분배 Todo
- 과목별 우선순위
- 공부 시간 시각화
- 집중 루틴 추천


[시나리오 B - 장기/단기 목표 사용자]

상황:
- 장기 목표 + 단기 할 일 함께 관리 원함

문제:
- 장기 목표가 묻힘
- 서로 성격 다른 Todo 혼합됨

필요 기능:
- 장기 / 단기 Todo 분리

단기:
- 즉시 실행 중심
- 체크형

장기:
- 습관
- 성장 추적
- 주간 / 월간 관리


6. 핵심 방향

To Do Dist =
단순 Todo 앱 X

“동기 유지 + 감정 기반 생산성 시스템”

- 작은 공간 점유
- 작업 방해 최소화
- 감정 기반 인터랙션
- 장기/단기 목표 분리
- 동기 유지 UX
- 공부 / 자기관리 특화
